Challenge 1: Name Your Creature

Find this line in the code:

let creatureName = 'Forest Spirit';

Change 'Forest Spirit' to any name you like

Try: 'Navi' or 'Kodama'

Challenge 2: Change the Glow Color

Find this line:

let creatureColor = '#00d4ff';

Try changing it to one of the hex colours below:

'#ff006e' for pink/magenta

'#00ff88' for green

'#ffaa00' for orange

'#aa00ff' for purple

Challenge 3: Add Wings

Find this line:

let hasWings = false;

Change false to true

Challenge 4: Add a Tail Trail

Find this line:

let hasTail = false;

Change false to true

Challenge 5: Change the Size

This time look in styles.css and find the section with the heading:

/* Size variation */

Inside this section find the following line:

transform: scale(1);

Try changing the scale(1) to:

scale(0.4) for a small creature

scale(1.8) for a big creature

Test out difference values to alter its size

Challenge 6: Add Glow Rings

Find this line:

let showGlow = false;

Change false to true
