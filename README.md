# Forest Spirit Workshop

An interactive web project where you can code and customize your own spirit creature

## Getting Started

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Edit `javascript.js` and `styles.css` to complete the coding challenges described in the app.

## Customization Challenges

- Change the creature's name and color in `javascript.js`
- Toggle wings, tail, and glow rings
- Adjust size in `styles.css`
- Add a click event for bonus interactivity

## Requirements

- Modern web browser
- No installation needed
